🔐 Encryption Tool
A simple and efficient encryption-decryption program built using Python. This project demonstrates how basic cryptographic techniques can be implemented to securely encode and decode messages.
🚀 Features
Encrypt plain text into secure cipher text
Decrypt cipher text back to original message
Easy-to-use interface (CLI-based)
Lightweight and beginner-friendly
Helps understand basic encryption concepts
🛠️ Technologies Used
Python 3
Standard Python libraries
